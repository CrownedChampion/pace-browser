What is a Pace Addon? Pace doesn't run Chrome extensions — modern ones rely on a background service worker that can't run on Pace's engine, which is why they fail to load. A Pace Addon is a small folder with an addon.json that the browser drives directly: it can hide elements, inject content scripts, and block network requests — no service worker needed. Building one is easy; the format is documented in the project README. Addons installed from outside the official Shop show a security warning first, because an addon can read and change the pages you visit.
